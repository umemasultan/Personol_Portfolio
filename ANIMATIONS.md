## ⚡ Performance Optimizations

### Framer Motion Best Practices
- **useInView hook**: Animations trigger only when in viewport
- **once: true**: Animations run once to save resources
- **Hardware acceleration**: Using transform and opacity for smooth 60fps
- **Stagger delays**: Prevents animation overload
- **Reduced motion**: Respects prefers-reduced-motion setting

### Loading Strategy
- **Lazy loading**: Images load only when needed
- **Priority images**: Hero image loads with priority
- **Code splitting**: Components load on demand
- **Optimized animations**: No janky animations, all 60fps
